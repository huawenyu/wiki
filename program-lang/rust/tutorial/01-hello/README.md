# Steps

Make sure you already have rust dev env before start.

1. create a file hello.rs with code:

```rust
fn main() {
    // Statements here are executed when the compiled binary is called

    // Print text to the console
    println!("Hello World!");
}
```

2. build:
    $ rustc hello.rs

3. run:
    $ ./hello
